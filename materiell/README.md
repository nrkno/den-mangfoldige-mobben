# Kort med mobb-programmeringsmønstre

Kortstokken består av tre typer kort; `rolle`, `strategi` og `avsporing` som beskriver mønstre man kan erfare i en mobb.

Mønstrene kan enten oppstå tilfeldig og utilsiktet, eller brukes som verktøy for å forbedre mobben eller løse spesifikke problemer som mobben har.

| Type | Beskrivelse|
|-------|--------------|
| Rolle | En funksjon eller oppførsel en person kan ha i mobben. Det kan være formelle roller eller gjenspeile personlige egenskaper. |
| Strategi | En måte mobben kan arbeide eller agere på. |
| Avsporing | En måte å jobbe på som er uheldig for mobben og som hindrer det man ønsker å oppnå ved å jobbe i mobb |


## Kort

### Rolle

| Type  | Kortnavn                    | Beskrivelse                                                    |
|-------|-----------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Rolle | Navigatøren                 | En som gir retning til sjåføren. Det kan være én eller flere navigatører i mobben, og de kan bidra i mobben på ulike måter.                                                                                                                              |
| Rolle | Forskeren                   | En som leter etter informasjon som mobben trenger for å komme videre.                                                                                                                                                                                    |
| Rolle | Opptakeren                  | En som på vegne av mobben noterer ting som for eksempel domeneinformasjon og designbeslutninger.                                                                                                                                                         |
| Rolle | Fasilitatoren               | En som hjelper mobben med å holde fokus og hjelper til med å løse uenigheter. Sjåføren har typisk ikke denne rollen.                                                                                                                                     |
| Rolle | Sjåføren                    | Den som er ved tastaturet, og gjør navigatørenes intensjoner og tanker om til kode eller annet på maskinen.                                                                                                                                              |
| Rolle | Djevelens advokat           | En navigatør som inntar en kritisk rolle for å gjøre design eller løsning bedre.                                                                                                                                                                         |
| Rolle | Automasjonisten             | En som legger merke til ting som gjentar seg og foreslår forbedring eller automatisering. Det kan gjelde oppgaver som gjentar seg, gjentagende steg i en prosess, gjentatt kode som kan refaktoreres, eller lignende.                                    |
| Rolle | Reserven                    | En som ikke vil være fullverdig medlem av mobben, og sitter i nærheten/utkanten og jobber med sitt, men kan svare raskt om mobben trenger innspill.                                                                                                      |
| Rolle | Snuseren                    | En som påpeker når koden lukter og foreslår forbedringer. Det kan gjelde for lange linjer, komplekse strukturer, duplisering, dårlig navngivning etc.                                                                                                    |
| Rolle | Trafikkonstabelen           | En som foreslår nye måter å arbeide på og påpeker og forsøker å hjelpe når mobben ikke følger avtalte prosesser, måter å jobbe på eller når arbeidsmåter er overflødige.                                                                                 |
| Rolle | Nostalgikeren               | En som foreslår en rask retrospektiv.                                                                                                                                                                                                                    |
| Rolle | Antropologen                | En som loggfører teamhistorikk, fasiliterer refleksjon over nye mobb-vaner, passer på innfasing og opplæring av nye mobb-medlemmer.                                                                                                                      |
| Rolle | Petimeteren                 | En som passer på at mobben sjekker at tester passerer, at det skrives test før kode, og at det ikke refaktoreres når tester er røde.                                                                                                                     |
| Rolle | Med hjartet på rette staden | En som sier ifra når noe i mobben ikke føles bra med prosess, verktøy eller mellommenneskelige ting. Viser empati når man merker at andre ikke har det bra. Foreslår retro for ting som ikke fungerer bra, så teamet kan forstå og forbedre.&nbsp;&nbsp; |
| Rolle | Dans på roser               | En som sier ifra om ting som gir en glede i mobben, merker når andre er glade, og foreslår retro for noe bra som skjedde, for å undersøke hvordan det kan skrus opp.                                                                                     |
### Strategi

| Type     | Kortnavn                       | Beskrivelse                                                                                                                                                                         |
|----------|--------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Strategi | Utbrytergruppe                 | En eller flere personer bryter ut av mobben for å fullføre en rutineoppgave. Utbrytergrupper kan også utforske alternativer som mobben vurderer samlet etterpå.                     |
| Strategi | Hold munn                      | En navigatør lar midlertidig være å snakke, så andre navigatører kan slippe til. Kan brukes for å få i gang en treg mobb eller for å forhindre at en person dominerer i mobben.     |
| Strategi | Punktliste                     | Mobben bruker en oppgaveliste for å organisere oppgaver og velge hva som er neste oppgave.                                                                                          |
| Strategi | Tvungen rotasjon               | En person i mobben tar over som sjåfør etter å ha blitt bedt om det av en annen i mobben, eller ved bruk av tidtaker.                                                               |
| Strategi | Si hva jeg skal gjøre          | Sjåføren forsøker å få innspill fra navigatørene, gir invitasjon til å navigere.                                                                                                    |
| Strategi | Plankekjøring                  | Sjåføren jobber med en oppgave og har som mål å fullføre den så raskt og smertefritt som mulig. Ofte kombinert med at sjåføren navigerer seg selv.                                  |
| Strategi | Egenrådig sjåfør               | Teamet bryr seg ikke om å bruke timer eller roller i mobben. Alle kan bidra med ideer, også sjåføren. Sjåføren må gjerne implementere egne ideer, men må uttrykke dem klart først.  |
| Strategi | Implementer alt                | Dukker det opp flere løsningsforslag, ikke bruk tid på å finne ut hvilket som er best. Implementer og test alle.                                                                    |
| Strategi | Roter sjåfør sjeldnere         | Roter sjåfør sjeldnere enn dere roterer navigatør. Nyttig dersom det tar lang tid å rotere sjåføren.Dette gjelder når mobben kun bruker én navigatør om gangen.                     |
| Strategi | Alle remote                    | Om teamet har hybrid mobb med noen remote, flytt alle til hvert sitt rom slik at alle er “remote”.                                                                                  |
| Strategi | Ekspertsjåføren                | Mobben velger den som best kjenner verktøyet man bruker, og som raskest kan få implementert mobben sine forslag, til å være sjåfør.                                                 |
| Strategi | Eksperten sist                 | Den med best kunnskap er sist i sjåfør-rekka. Eksperten fungerer som navigatør lengst, og bidrar til kompetansedeling.Dette gjelder når mobben kun bruker én navigatør om gangen.   |
| Strategi | Smøre urverket                 | Om du som navigatør er usikker, spør om hjelp. Resten av mobben står klar med ideer.                                                                                                |
| Strategi | Hurtigmobben                   | Mobben gir sjåføren instrukser om hvordan oppgaven løses. Resten av mobben trekker seg unna for å planlegge neste oppgave.                                                          |
| Strategi | Loggføring                     | Opprett rollen loggfører, som går på rundgang. Før logg om alle diskusjoner og beslutninger. Teammedlemmer som er borte, kan lese loggen for å komme a jour.                        |
| Strategi | Ingen private maskiner         | Ingen personlige maskiner i mobb-sonen. Dette sørger for fokus på mobben sine oppgaver.                                                                                             |
| Strategi | Alle har privat maskin         | Alle har hver sin maskin i tillegg til mobb-maskina. Alle kan bidra med å lete etter gode løsninger underveis.                                                                      |
| Strategi | Minst erfarne sine ideer først | Når det dukker opp ulike ideer, prøv alltid ideene til den minst erfarne først.                                                                                                     |
| Strategi | Flere navigatører              | Alle som ikke er sjåfør, er navigatører. Samtalene flyter fritt, og man enes fort om løsningene.                                                                                    |
| Strategi | Mobodoro                       | Mobb i 25 minutter med korte intervall, kort retrospektiv og pause. Repeter.                                                                                                        |
| Strategi | Naviger navigatørene           | Eksperten forteller, på et overordnet plan, de øvrige navigatørene hva som skal skje. Navigatørene navigerer så mer detaljert basert på dette.                                      |
| Strategi | Én navigatør                   | Bare én navigerer sjåføren om gangen, og mobben bytter navigatør samtidig som den bytter sjåfør. De andre i mobben kan komme med innspill til navigatøren.                          |
| Strategi | Stol-leken                     | Alle bytter plass når man bytter sjåfør.                                                                                                                                            |
| Strategi | Nå skal jeg lære               | Når en i mobben er usikker eller ny, bruk av vedkommende sin navigatørtid til opplæring i stedet for koding.                                                                        |
| Strategi | Tørrtrening                    | Øv på stolleken for å bli vant til å rotere, og forstå at man ikke alltid skal ha samme rolle.                                                                                      |
| Strategi | Sentrifuge                     | Roter raskt, øk gradvis rotasjonstempoet, gjerne ned til ett minutt. Ha mikro-retro for å få frem erfaringer.                                                                       |
| Strategi | Snu retningen                  | Om man normalt blir navigatør umiddelbart etter å ha vært sjåfør, eller motsatt, bytt om på dette for å erfare noe nytt.Dette gjelder når mobben kun bruker én navigatør om gangen. |
| Strategi | Rollespill                     | Har mobben roller som ikke blir fylt opp godt nok, prøv rollespill.                                                                                                                 |
| Strategi | Faste plasser                  | Når du ikke lenger er sjåfør, flytt deg tilbake til din faste plass. Ikke stolleken.                                                                                                |
| Strategi | Fast sjåfør                    | Alltid samme person som er sjåfør.                                                                                                                                                  |
| Strategi | Roter ved hendelser            | Ikke bruk timer, men roter når noe er fullendt. For eksempel når en test er laget, eller en test går grønn.                                                                         |
| Strategi | Bruk timer                     | Bruk en timer, og vær streng med å bytte umiddelbart (ikke noe “skal bare”).                                                                                                        |
| Strategi | Oppsummering                   | Ved hvert bytte forteller sjåfør eller navigatør hva som nettopp ble gjort, og hvorfor. Det hjelper den neste å fortsette der den forrige slapp.                                    |
| Strategi | Pause                          | Ta pauser, jevnlig eller ved behov.                                                                                                                                                 |
| Strategi | Fast navigatør                 | Alltid samme person som navigerer, men navigatøren får innspill fra resten av mobben.                                                                                               |
| Strategi | Smart tastatur                 | Sjåføren er et smart tastatur som får instrukser, på et høyest mulig abstraksjonsnivå, og utfører uavhengig av egne meninger.                                                       |
| Strategi | Tenk høyt                      | Bidra med, og sett pris på halvtenkte tanker. Spinn videre på ideene i fellesskap.                                                                                                  |
| Strategi | Skattekartet                   | Når målet er uklart, finn frem tavla, og tegn og tenk i fellesskap.                                                                                                                 |

### Avsporing

| Type      | Kortnavn           | Beskrivelse                                                                                                                                |
|-----------|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| Avsporing | Løs kanon          | En navigatør dikterer på egenhånd mobbens arbeid og ekskluderer de andre navigatørene.                                                     |
| Avsporing | Boksekamp          | Situasjon der to eller flere deltakere krangler om retningen mobben skal ta, uten å bry seg om prinsipper om gjensidig respekt og omtanke. |
| Avsporing | Ufokusert deltaker | Person er navigatør i mobben, men uten å delta aktivt. Muligens distrahert av andre ting.                                                  |
| Avsporing | Tenke høyt         | Sjåføren tenker høyt fordi vedkommende er den som kan mest om oppgaven eller ser hvordan man kan komme videre.                             |
| Avsporing | På autopilot       | En sjåfør fortsetter å skrive uten å ha fått innspill fra resten av mobben.                                                                |
